The backend setup uses the Python files you provided (views.py, serializers.py, etc.).

1. Install Dependencies:
   In the VS Code terminal, run the following command to install all necessary Python packages:
   ```cmd
   pip install -r requirements.txt
   ```

2. Navigate to Backend:
   Change directory into the backend folder:
   ```cmd
   cd backend
   ```

3. Apply Migrations:
   Run the Django migration commands to create the necessary database tables:
   ```cmd
   python manage.py makemigrations
   python manage.py migrate
   ```

4. Run Backend Server:
   Start the Django development server. Keep this terminal window open and running.
   ```cmd
   python manage.py runserver
   ```

   The backend should now be running, typically at:
   ðŸ‘‰ http://127.0.0.1:8000/
